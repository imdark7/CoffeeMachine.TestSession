Файл генерируется заново при каждом локальном прогоне задания. Не забывайте коммитить!
======
Incorrect_RunA
Всегда рассчитывает стоимость как за латте

======
Incorrect_RunB
Позволяет приготовить кофе, когда он уже готов

======
Incorrect_RunC
Можно сделать кофе, если не выбраны Size и/или SugarLevel

======
Incorrect_RunD
Готовит всегда Латте

======
Incorrect_EjectCupA
Size не сбрасывается после приготовления

======
Incorrect_EjectCupB
Кофе можно забрать, когда он еще не готов

======
Incorrect_GetChangeA
Баланс не меняется, если забирать сдачу при статусе TakeYourCoffee

======
Incorrect_GetChangeB
Сбрасывается статус работы, если забрать сдачу прежде чем забрали готовый кофе

======
Incorrect_SelectSizeA
Нельзя выбрать маленький размер чашки

======
Incorrect_SelectSizeB
Можно приготовить кофе не выбрав уровень сахара

======
Incorrect_SelectSizeC
Можно перевыбрать размер чашки только на большую

======
Incorrect_InsertMoneyA
Можно закинуть деньги, пока не достал готовый кофе

======
Incorrect_InsertMoneyC
Можно закинуть деньги только после выбора кофе

======
Incorrect_InsertMoneyE
Можно перейти к кастомизации кофе, даже если не хватает денег

======
Incorrect_SelectSugarA
Можно перевыбрать количество сахара, когда кофе уже готов

======
Incorrect_SelectSugarB
Нельзя перевыбрать уровень сахара

======
Incorrect_SelectCoffeeA
Можно перевыбрать кофе после его приготовления

======
Incorrect_SelectCoffeeB
Можно выбрать тип кофе Unknown

======
Incorrect_SelectCoffeeC
Можно приготовить кофе, если не хватает денег на него

======
IncorrectCoffeeMachineAlwaysFails
Падает при вызове любого метода

======
Incorrect_GetChangeC
Секретный баг
Если три раза вернуть ненулевую сдачу, то на третий раз вернется 1000

======
Incorrect_GetChangeD
Секретный баг
Не сбрасывается Уровень сахара после того, вернули сдачу

======
Incorrect_InsertMoneyB
Секретный баг
Можно ввести отрицательное значение денег, если баланс не уйдет ниже 0

======
Incorrect_InsertMoneyD
Секретный баг
Если кофе выбран, при вводе денег к балансу добавляется стоимость кофе, а не введеннная сумма

======
Incorrect_InsertMoneyF
Секретный баг
Если вводить деньги по одному рублю, то можно приготовить любой кофе всего за 10 рублей

======
Incorrect_InsertMoneyG
Секретный баг
Баланс пополняется, только если вводить разные суммы

======
Incorrect_SelectSugarC
Секретный баг
Если выбрать количество сахара High, то размер чашки меняется на Large
